==== insert an svg file ====
tags | show svg file

  * Type '!' then select the attached SVG
  * It renders inline
  * Adjust the image size as needed.

Ref:- https://bobswift.atlassian.net/wiki/spaces/info/pages/31916055/How+to+display+SVG+attachments+in+Confluence
