==== outlook calendar not showing all times ====
<code>
File -> Options -> Calendar
-> Work time -> Work Hours
-> set 'Start time:' to 12:00 AM
-> Set 'End time:' to 11:30 PM
-> Select all days in 'Work Week:'
-> OK
</code>
