==== ankidroid.A and ankidroid.B setup ====
tags | ankidroid A, ankidroid B

Open AnkiDroid.A app -> Settings -> Advanced -> set AnkiDroid directory to /storage/emulated/0/AnkiDroid.A

Open AnkiDroid.B app -> Settings -> Advanced -> set AnkiDroid directory to /storage/emulated/0/AnkiDroid.B

Optional:

Delete any files in stored in the default directory /storage/emulated/0/AnkiDroid

Open the file manager -> Internal storage -> delete the AnkiDroid folder.

Ref:
  * https://github.com/ankidroid/Anki-Android/wiki/FAQ#how-to-use-different-anki-profiles
    * > you can achieve a similar effect by installing several parallel versions of AnkiDroid besides the normal AnkiDroid and ''setting a unique collection path in each one''
  * https://github.com/ankidroid/Anki-Android/releases
    * > Parallel builds install side-by-side with the main APK, allowing you to use different settings and profiles (''via the AnkiDroid directory advanced setting and a different AnkiWeb login'').

==== remove duplicate cards ====
I am using the "Remove Duplicate Cards" addon available at https://ankiweb.net/shared/info/95590040 .

To install it
<code>
start anki on the command line
-> Tools -> Add-ons
-> Get Add-ons
-> enter the code displayed on the addon page (ex:- 95590040)
</code>

Ref:-
  * https://docs.ankiweb.net/addons.html
  * https://ankiweb.net/shared/info/95590040

==== AnkiDroid directory is inaccessible ====

Settings -> Security & privacy -> Privacy -> Permission manager -> Files -> "See more apps that can access all files" -> AnkiDroid.A -> Allow access to manage all files -> drag to the right.
