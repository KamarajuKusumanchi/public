===== configuration =====
==== disable email notifications ====
  * Settings -> Notifications and activity -> Missed activity emails -> Off

==== open links in Chrome ====
  * Settings -> Files and links -> Links open preferences -> Always open web links in: -> choose "Default browser" in the dropdown.

Then change the default browser at the system level.
